O que é estilização de páginas?

Estilização de paginas é um meio de se comunicar a partir das cores e elementos como imagem, videos, etc. Sendo uma das partes mais importantes para a caracterização do seu site.

O que é semantica da programação W3C?

É a linha do código onde se encontra as tags solicitadas dentro do mesmo código, por exemplo, a tag style entre outras.

Tipos de Seletores

São selecionadores de itens do CSS, seus tipos são Select id(pode ser utilizado para um unico elemento), classe( pode ser utilizado por varios elementos) e elementos(Pode ser utilizado em aenas um elemento nomeado)
